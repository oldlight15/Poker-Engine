# Python Texas Hold'em Engine & AI

A modular Texas Hold'em simulation engine built in Python. The core focus of this project is implementing a data-driven AI that utilizes Monte Carlo simulations to make mathematically optimal betting decisions.

## Current Status: In Progress
- [x] Core OOP architecture (Cards, Decks, Players)
- [x] Hand evaluation logic (Pairs, Flushes, Straights)
- [x] Monte Carlo simulation skeleton
- [ ] Implement varied AI profiles (Tight-Aggressive, Tight-Passive, Loose-Passive)
- [ ] Full betting round game loop
